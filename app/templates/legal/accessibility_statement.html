Accessibility Statement

Last updated: 14/04/2025

Introduction

RevisePDF is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

RevisePDF is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

RevisePDF includes the following accessibility features:

  • Semantic HTML structure with appropriate headings and landmarks
  • Sufficient colour contrast for text and interface components
  • Text alternatives for non-text content
  • Keyboard accessibility for all functionality
  • Clear focus indicators for keyboard navigation
  • Responsive design that adapts to different screen sizes and orientations
  • Form labels and error messages that are programmatically associated with their controls

Known Limitations

Despite our best efforts to ensure accessibility, there may be some limitations:

  • Some PDF documents generated by our tools may not be fully accessible, depending on the source content
  • Some interactive elements may not be fully accessible via keyboard navigation
  • Some form controls may not have properly associated labels

We are working to address these issues and improve the accessibility of our website.

Assessment Approach

RevisePDF assessed the accessibility of this website by the following approaches:

  • Self-evaluation using automated testing tools
  • Manual testing with assistive technologies
  • User testing with individuals with disabilities

Compatibility with Browsers and Assistive Technology

RevisePDF is designed to be compatible with the following assistive technologies:

  • Screen readers (including NVDA, JAWS, and VoiceOver)
  • Screen magnifiers
  • Speech recognition software
  • Keyboard-only navigation

RevisePDF is compatible with recent versions of major browsers, including Chrome, Firefox, Safari, and Edge.

Technical Specifications

Accessibility of RevisePDF relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript
  • WAI-ARIA

These technologies are relied upon for conformance with the accessibility standards used.
